## 🔐 Security Philosophy

```
┌─────────────────────────────────────────────────────────────────┐
│  SECURITY-FIRST ARCHITECTURE PRINCIPLES                         │
├─────────────────────────────────────────────────────────────────┤
│  ✅  Least privilege IAM  →  Zero standing access              │
│  ✅  Encrypt at rest      →  KMS for all sensitive data        │
│  ✅  Encrypt in transit   →  TLS 1.2+ enforced everywhere      │
│  ✅  Audit everything     →  CloudTrail + CloudWatch Logs      │
│  ✅  Shift left security  →  SAST/DAST in CI/CD pipelines      │
│  ✅  Immutable infra      →  No SSH, use SSM Session Manager   │
│  ✅  Secrets rotation     →  Automated via Secrets Manager     │
└─────────────────────────────────────────────────────────────────┘
```
